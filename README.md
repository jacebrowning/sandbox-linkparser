# sandbox-linkparser

[![Build Status](https://travis-ci.org/jacebrowning/sandbox-linkparser.svg?branch=master)](https://travis-ci.org/jacebrowning/sandbox-linkparser)

## Requirements

* Python 3.6.0
* pipenv: 3.5.4
* NPM 3.10.8

## Usage

Install dependencies:

```sh
$ make install
```

Run the application:

```sh
$ make run
```

Run the tests:

```sh
$ make test
```

Keep the tests running:

```sh
$ make watch
```
